# 個室サウナ エリア別 取りこぼし調査

指定エリアの個室サウナを網羅的に調査し、未登録施設を洗い出す。

## 引数

$ARGUMENTS: 調査対象エリア（例: 「東京」「大阪」「横浜」）

## 手順

### Phase 1: 既存施設リストの取得

`data/facilities.json` から対象エリアの登録済み施設を抽出し、施設名リストを作成する。

### Phase 2: 多角的な検索（並列実行）

以下の検索を **3つのサブエージェント** で並列実行する。
**重要**: 検索時点では候補を広く集め、Phase 3 で厳密に裏取りする。

#### Agent A: 厳密キーワード検索
以下のキーワードで WebSearch:
- 「{エリア} "完全個室" サウナ」
- 「{エリア} "プライベートサウナ" 個室」
- 「{エリア} "1組貸切" サウナ」
- 「{エリア} 個室サウナ 一覧 {current_year}」

#### Agent B: 新規オープン + PR Times
以下のキーワードで WebSearch:
- 「{エリア} 個室サウナ 新規オープン {current_year}」
- 「{エリア} 個室サウナ オープン {current_year - 1}」
- 「site:prtimes.jp {エリア} 個室サウナ」
- 「site:prtimes.jp {エリア} プライベートサウナ オープン」

#### Agent C: 予約サイト + 系列店チェック
- 「site:coubic.com 個室サウナ {エリア}」
- 「site:reserva.be プライベートサウナ {エリア}」
- 既存施設の **系列店** を公式サイトで確認（例: KUDOCHI → 他店舗、HUBHUB → 他店舗）
- 既存施設と同じビル・エリアに新店がないか確認

### Phase 3: 個室サウナ裏取り（最重要）

Phase 2 で集めた候補を **1施設ずつ公式サイトで裏取り** する。
**「集める」と「裏取り」を同時に行うこと。後回しにしない。**

#### 判定基準

| 判定 | 条件 |
|------|------|
| ○ 個室サウナ | サウナ室が完全個室（1組貸切）。他の客と空間を共有しない |
| △ 貸切プランあり | 通常は共用だが、施設全体または一部を貸切にするプランがある |
| × 個室ではない | 大浴場型・共用サウナ室。複数組が同時利用する |

#### 裏取りチェックポイント

| チェック項目 | 個室サウナの特徴 | 共用サウナの特徴 |
|-------------|-----------------|-----------------|
| 料金体系 | 「1室あたり」「1組あたり」 | 「1人あたり」「入館料」 |
| 予約方式 | 完全予約制・時間枠指定 | 予約不要・先着順 |
| 収容人数の表記 | 「最大◯名」（少人数） | 「定員◯名」（大人数）or 記載なし |
| 設備配置 | 各室にサウナ・水風呂・休憩が一体 | サウナ室→水風呂→休憩スペースが別 |
| サイトの表現 | 「完全個室」「プライベート」「貸切」 | 「男性専用」「ロウリュイベント」 |

#### よくある誤検知パターン

| パターン | 例 | 対処 |
|----------|-----|------|
| 「プライベート感」を強調する共用サウナ | 「プライベート空間のような...」 | 料金体系（1人あたり）で見抜く |
| 個室休憩はあるがサウナ室は共用 | スーパー銭湯の個室休憩プラン | サウナ室自体が個室かを確認 |
| ホテルの大浴場サウナ | 「客室でサウナ気分」 | 客室内にサウナ室があるか確認 |
| 閉店済み施設 | まとめ記事に残っている | 公式サイト・SNSで営業状況を確認 |
| 同一施設の別名 | 「上野プライベートサウナ」= RED° E-SAUNA UENO | 住所で既存施設と照合 |

### Phase 4: 結果の整理

以下の形式でテーブルにまとめる:

#### 未登録の個室サウナ（確認済み）

| 施設名 | エリア | 料金目安 | note | 公式URL |
|--------|--------|----------|------|---------|
| ... | ... | ... | null or "会員制/ビジター可" | ... |

#### 会員制施設

会員制の場合は以下を確認して記載:
- ビジター利用可能か
- ビジター料金
- → 可能なら `note: "会員制/ビジター可"` で登録候補に含める
- → ビジター不可なら登録対象外として報告のみ

#### 未オープン（ウォッチリスト）

| 施設名 | エリア | オープン予定日 | 公式URL |
|--------|--------|---------------|---------|

#### 除外した施設

| 施設名 | 除外理由 |
|--------|----------|
| ... | × 共用サウナ / 閉店済み / 既登録の別名 |

### Phase 5: 追加の案内

未登録施設がある場合:

```
追加したい施設があれば `/add-facility <施設のWebサイトURL>` で追加できるよ!
まとめて追加する場合は `/sauna-scout` も使えるよ!
```

## 注意事項

- **裏取りファースト**: 候補を集めたら即座に公式サイトで個室かどうか確認する。「後でまとめて確認」は共用サウナの混入を招く
- **系列店チェック**: 既存施設のブランド（KUDOCHI, HUBHUB, Prus Sauna 等）が新店舗を出していないか必ず確認
- **閉店チェック**: まとめ記事の情報は古い可能性がある。公式サイト・SNSで営業中か確認
- **sauna-ikitai.com は使用禁止**（ブロックされる + ユーザー方針）
