# 個室サウナ 新着ニュース収集

新しい個室サウナのオープン情報を収集し、未登録施設をハイライトする。

## 手順

### 1. 最新ニュースを検索

WebSearch で以下のキーワードを**並列で**検索する（現在の年を使用）:

- 「個室サウナ オープン {current_year}」
- 「プライベートサウナ 新店 {current_year}」
- 「個室サウナ 新規オープン {current_year}」
- 「site:prtimes.jp サウナ 個室 {current_year}」
- 「site:sauna-ikitai.com 個室サウナ」

### 2. 既存施設と照合

`data/facilities.json` を読み込み、既存の施設名リストを取得する。

### 3. 結果を整理

以下の形式でテーブルにまとめる:

| 状態 | 施設名 | エリア | オープン日 | URL |
|------|--------|--------|-----------|-----|
| ⭐ NEW | （未登録施設） | ... | ... | ... |
| ✅ 登録済 | （既存施設） | ... | ... | ... |

- 施設名が `facilities.json` に存在しない場合は「⭐ NEW」マーク
- 施設名が既存の場合は「✅ 登録済」マーク
- 曖昧なマッチ（名前が微妙に違う等）は「❓ 要確認」マーク

### 4. 案内

未登録施設がある場合、以下のように案内する:

```
気になる施設があれば `/add-facility <施設のWebサイトURL>` で追加できるよ!
```

## 注意事項

- 「個室サウナ」に該当しない施設（公衆サウナ、スパ施設のサウナコーナー等）は除外する
- 情報の信頼性が低い場合はその旨を明記する
- 検索結果が多い場合は直近3ヶ月以内のニュースを優先する
