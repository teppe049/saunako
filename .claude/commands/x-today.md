# X 今日の投稿テーマ確認

今日やるべきX投稿を確認し、実行する。

## 手順

### 1. カレンダー確認

`docs/x-drafts.md` の投稿カレンダーから今日の日付のテーマを取得。

### 2. 表示

以下のフォーマットで報告:

```
## 今日のX投稿（M/D）

| # | 種別 | テーマ | 切り口 | 状態 |
|---|------|--------|--------|------|
| 1 | 画像まとめ | ○○ | ○○ | — |
| 2 | 軽テキスト | ○○ | ○○ | — |
```

### 3. 実行提案

- 画像まとめがある日 → 「`/x-list-post {テーマ}` で始める？」
- 軽テキストがある日 → 投稿文の案を3つ提示
- 両方ある日 → 画像まとめを先にやることを提案

### 4. 前日の振り返り（簡易）

前日の投稿のimp・いいね・プロフ訪問を確認できる場合は簡易報告。
（X Analyticsのデータが取れない場合はスキップ）

### 5. カレンダーに未設定の日

投稿カレンダーに今日のテーマがない場合:
1. `docs/x-strategy.md` の切り口ローテーションを確認
2. 直近の投稿テーマを `docs/x-post-log.md` で確認（同じ切り口の連打を避ける）
3. テーマ候補を3つ提案 → ユーザーに選んでもらう
