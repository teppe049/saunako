# サウナ子 キャラクターガイドライン

Saunako プラットフォーム上のテキスト（施設コメント・エリア紹介・UIセリフ）を統一するためのガイドライン。
コメントやUIテキストを追加・編集する際は、必ずこのドキュメントに準拠すること。

---

## 基本設定

| 項目 | 内容 |
|------|------|
| **名前** | サウナ子 |
| **性格** | サウナが大好きな友達。明るくて親しみやすい |
| **一人称** | 使わないか「私」 |
| **語尾** | だよ / かも / ね / よ / ! |
| **立場** | ユーザーと同じサウナ好きの目線で、施設の魅力を伝える |

---

## 話し方ルール

### 施設コメント（Short: 20〜40文字）

施設の最大の特徴を端的に伝える一文。検索結果カードなどで使用。

- 1つの施設につき1つの魅力に絞る
- 体言止めや「!」で歯切れよく
- ユーザーが「行ってみたい」と思える情報を優先

**例:**
- 「3時間貸切でグループ利用に最適!」
- 「コスパ抜群で駅チカなのが嬉しいポイント!」
- 「深夜でも整えるのが最高だよ!」
- 「本格フィンランド式サウナを独り占めできるよ!」

### 施設コメント（Long: 80〜200文字）

施設の魅力を具体的に紹介するテキスト。施設詳細ページで使用。

- 水風呂の温度、ロウリュ（セルフ/オート）、外気浴の有無など、サウナ好きが気になるポイントに言及
- 施設の雰囲気やロケーションにも触れる
- 1文目で最大の魅力を伝え、後半で補足情報を追加

**例:**
> 銀座で24時間いつでも整えるのが嬉しいポイントだよ! 水風呂もしっかり冷えるし、セルフロウリュもできるから自分好みの蒸気を楽しめるかも。アクセスも抜群だから、仕事帰りにふらっと寄れるのもいいよね!

### エリアコメント（area ページ: 100〜150文字）

そのエリアの個室サウナの特徴を紹介するテキスト。エリア一覧ページの冒頭で使用。

- エリアの個室サウナの傾向（件数・価格帯・特徴）に触れる
- 「このエリアならではの魅力」を伝える

**例:**
> 渋谷エリアには駅チカの個室サウナがたくさんあるよ! 仕事帰りやショッピングのついでに立ち寄りやすいのが魅力だね。おしゃれな空間が多いから、サウナ初心者の友達を誘うのにもぴったりかも!

### UIセリフ（トップページ等）

ユーザーへの呼びかけ。ボタン周辺やヒーローセクションなどで使用。

- 短く親しみやすく（15〜30文字程度）
- 行動を促すニュアンスを含める

**例:**
- 「今日はどこで整える?」
- 「あなたにぴったりのサウナを見つけよう!」
- 「気になるサウナをチェックしてみてね!」

## NG表現

### 敬語禁止

サウナ子は「友達」の立場なので、敬語は使わない。

| NG | OK |
|----|-----|
| 「おすすめです」 | 「おすすめだよ」 |
| 「ございます」 | 「あるよ」 |
| 「いかがでしょうか」 | 「どうかな?」 |

### 断定しすぎ禁止

個人の感想として伝える。強すぎる断定はしない。

| NG | OK |
|----|-----|
| 「絶対行くべき」 | 「ぜひ行ってみてほしいな」 |
| 「必ず満足する」 | 「きっと気に入ると思うよ」 |
| 「間違いなく最高」 | 「最高かも!」 |

### その他NG

- **施設への否定的なコメント** -- どの施設にもそれぞれの良さがある前提で書く
- **他施設との比較** -- 「A施設よりB施設のほうが〜」のような表現は禁止
- **価格の具体的な言及** -- 料金は変動するため、「コスパがいい」程度に留める

## サンプル集（Good / Bad 対比）

### 1. 施設コメント Short

| | テキスト | 理由 |
|------|----------|------|
| Good | 「深夜3時まで営業で仕事帰りにも寄れるよ!」 | 具体的な特徴 + カジュアルな語尾 |
| Bad | 「深夜3時まで営業しております」 | 敬語、事実の羅列で魅力が伝わらない |

### 2. 施設コメント Long

| | テキスト | 理由 |
|------|----------|------|
| Good | 「水風呂が14度でキンキンに冷えてるのが最高! セルフロウリュで好きなだけ蒸気を浴びられるし、外気浴スペースもあるから整い環境はバッチリだよ」 | サウナ好き視点の具体情報 + 友達口調 |
| Bad | 「水風呂は14度です。セルフロウリュが可能です。外気浴スペースがございます」 | 敬語、箇条書き的で感情がない |

### 3. エリアコメント

| | テキスト | 理由 |
|------|----------|------|
| Good | 「新宿エリアは深夜営業の個室サウナが充実してるよ! 終電を逃しても安心だし、朝までゆっくり整えるのが嬉しいポイントだね」 | エリアの特徴 + ユーザーメリット |
| Bad | 「新宿エリアには個室サウナが5店舗あります。深夜営業の店舗もございます」 | 敬語、データ的で魅力が伝わらない |

### 4. UIセリフ

| | テキスト | 理由 |
|------|----------|------|
| Good | 「今日はどこで整える?」 | 短い、親しみやすい、行動を促す |
| Bad | 「お好みのサウナ施設をお探しください」 | 敬語、堅い、距離感がある |

### 5. 施設の特徴紹介

| | テキスト | 理由 |
|------|----------|------|
| Good | 「ここは絶対…じゃなくて、めちゃくちゃおすすめかも! アロマロウリュの香りがすごく良くて、リラックスできるよ」 | 柔らかい表現、具体的な体験 |
| Bad | 「この施設は絶対に行くべきです。他の施設より断然優れています」 | 断定しすぎ、敬語、他施設比較 |
