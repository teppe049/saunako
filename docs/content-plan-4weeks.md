## CTA導線改善（3/29策定、3/30〜実施）

### 課題
views 93/投稿なのにOrganic Social 14PV/週（転換率1.1%）。目標30PV/週。

### 変更点（3/30〜全投稿に適用）

**本文構成:**
```
[フック（知ってた？/実は○○）]

[施設の特徴・設備の紹介]
※料金は本文に書かない

で、肝心の料金なんだけど——

📍エリア
#個室サウナ #サウナ
```

**リプ構成:**
```
施設名
💰 料金情報（1人あたり計算も）
📍 最寄り駅 徒歩○分
→ 同エリアの他の個室サウナも比較できるよ
https://www.saunako.jp/facilities/{id}?utm_source=x&utm_medium=social
```

**変更の狙い:**
- 本文から料金を外し「で、肝心の料金なんだけど——」で引きを作る → リプ到達率↑
- リプに料金+アクセス+比較導線を入れる → クリック動機↑
- 全URLにUTMパラメータ付与 → GA4でパターン別計測可能に

### 測定（毎日記録）
- 本文 views（X Analytics / Playwright実測）
- リプ views → **リプ到達率 = リプviews / 本文views**
- Organic Social PV（GA4、週次）

### 判定（4/5 Week4レビュー）
- リプ到達率 > 20% かつ Organic Social ≥ 20PV/週 → 成功、継続
- リプ到達率 < 20% → 本文の引き方を改善（情報ギャップ型等にテスト）
- リプ到達率 > 30% だがPV横ばい → リプ文面をさらに改善
