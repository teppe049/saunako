# E-E-A-T 強化プレイブック

個室サウナに頻繁に行けなくても実践できるE-E-A-T（Experience, Expertise, Authoritativeness, Trustworthiness）強化施策。

## 背景

- 2026/3/11以降、Googleの品質フィルターにより検索トラフィックが消滅
- 128施設のseoTitle一括変更 × アルゴリズム変動が原因と推定
- Google は「AIが作っただけのサイト」を低評価する傾向を強化中
- 回復には E-E-A-T シグナルの強化が必要

---

## 1. データ分析記事（すぐできる・高コスパ）

393施設のデータを持っているからこそ書ける一次情報。

### 記事ネタ例

- 「393施設を調べてわかった個室サウナの平均料金」
  - 都道府県別・エリア別の料金比較
  - 最安/最高施設ランキング
- 「水風呂の温度は平均○℃。チラー付き施設の割合は？」
- 「カップルOK率は○%。意外と少ない○○な設備」
- 「2026年に新しくオープンした個室サウナまとめ」（openedAtデータ活用）
- 「60分以下で入れるコスパ最強施設」

### 実装方法

```bash
# facilities.json を集計するスクリプトを書いて分析
# → 結果を記事MDXに落とし込む
```

## 2. オーナー取材（DM/メールでOK）

実際に訪問しなくても、オーナーの声 = 強力な一次情報。

### 手順

1. X or Instagram で施設公式アカウントにDM
   - 「サウナ子（saunako.jp）というサイトで紹介させていただいています」
   - 「こだわりポイントや、お客様の声を教えていただけませんか？」
2. 回答を記事に引用（「オーナーに聞きました」セクション）
3. 施設ページの `note` や `saunakoCommentLong` にも反映

### テンプレ文

```
はじめまして。個室サウナ検索サイト「サウナ子」(saunako.jp)を運営しております。
○○様の施設を掲載させていただいておりますが、
より魅力をお伝えするため、以下を教えていただけますと幸いです。

1. 施設のこだわりポイント
2. お客様から好評なポイント
3. おすすめの利用シーン

お忙しいところ恐れ入りますが、ご協力いただけますと幸いです。
```

### 期待効果

- 施設ページの情報量UP → Google評価向上
- オーナーとの関係構築 → 被リンク獲得の布石
- 相互リンクの打診にもつなげられる

## 3. SNS口コミの集約

X・Google Maps・Instagram の利用者レビューを引用。

### 手順

1. X で施設名を検索 → 実際の利用者の投稿を見つける
2. 施設ページに「利用者の声」セクションを追加
3. 出典（投稿URL等）を明記して引用

### 注意点

- 引用は出典を明記（著作権配慮）
- ネガティブな口コミは載せない（施設との関係を壊さない）
- 定期的に更新する（古い口コミは外す）

## 4. サウナ子のキャラクター活用

「全部行った人」ではなく「誰よりも調べた人」として確立。

### やること

- `/about` ページに運営者情報を追加
  - 「全国393施設の個室サウナを徹底調査」
  - 「料金・設備・口コミを比較して、あなたにぴったりの1軒を見つけます」
  - 調査方法の透明性（公式サイト・予約サイト・SNSを確認）
- サウナ子コメントのテンプレ感を減らす
  - 施設ごとの具体的な推しポイントを入れる
  - 「ここの水風呂は○℃で、3セット目がちょうどいい」等の具体性

---

## 5. 訪問レポート（行けたときに最大活用）

たまに行けた施設は1施設で1記事書く。3〜5本あればサイト全体の信頼性が上がる。

### 記事構成

1. 予約〜到着までの流れ
2. 施設の第一印象（写真）
3. サウナ室・水風呂・外気浴の詳細レビュー
4. 良かった点・気になった点
5. こんな人におすすめ
6. 料金・アクセスまとめ

### ポイント

- 写真は10枚以上（スマホでOK）
- 「実際に行ってみた」タグを作って記事をまとめる
- 施設ページからも「訪問レポートはこちら」でリンク

## SEO変更の段階的実施ルール

CLAUDE.md にも記載済み。

- seoTitle / seoDescription の変更は **1回のデプロイにつき最大10施設**
- 変更後は **1〜2週間あけてから** 次のバッチを実施
- `/analytics-pdca` で Google SC の imp/clicks を確認してから次に進む
