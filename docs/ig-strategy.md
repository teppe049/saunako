# サウナ子 Instagram運用戦略

## ポジショニング

X が「会話・意見」、Instagram は「保存・発見」。
カルーセルは保存率が高くアルゴリズムに有利。ビジュアル重視の「保存される」コンテンツでロングテール流入を狙う。

### Xとの役割分担

| | X | Instagram |
|---|---|-----------|
| 役割 | 会話・比較・意見 | 保存・発見・ビジュアル |
| 強み | 拡散力・即時性 | 保存率・検索性・視覚訴求 |
| コンテンツ | テキスト中心 | 画像カルーセル中心 |
| CTA | プロフのリンクへ | プロフのリンクから |
| ハッシュタグ | 2個 | 10-15個 |

## ハッシュタグ戦略（10-15個/投稿）

3層構造でリーチとニッチを両立。

| Tier | 個数 | 例 | 目的 |
|------|:----:|-----|------|
| Tier1（固定） | 3-5 | `#個室サウナ` `#サウナ` `#サウナ好きな人と繋がりたい` | コア層へのリーチ |
| Tier2（地域） | 3-5 | `#東京サウナ` `#渋谷` `#名古屋サウナ` | 地域検索の発見 |
| Tier3（広域） | 2-3 | `#ととのう` `#サ活` `#デートスポット` | 非サウナ層への拡張 |

- ハッシュタグはキャプション末尾に配置
- テーマに応じてTier2・Tier3を入れ替える

## 成長戦略

### Month 1: ポートフォリオ構築

- カルーセル4投稿 + 施設紹介4投稿 = 計8投稿
- Xからクロスプロモ（「画像でまとめたよ! IGもチェックしてね」）
- ハッシュタグの効果検証

### Month 2-3: 発見フェーズ

- ハッシュタグ経由の自然流入を増やす
- 施設アカウントとの相互タグで露出拡大
- 保存率が高いテーマを特定

### Month 4+: 集中フェーズ

- 保存数が高いテーマに集中投稿
- フォロワー300目標
- IG経由のサイト流入を月間目標に設定

---

## Xとのクロスプロモ

- カルーセル投稿日に X で「画像でまとめたよ! IGもチェックしてね」
- IG カルーセルのテーマを翌日 X でテキスト議論
- IGのプロフリンクからサイトへ → サイト内でX誘導も可能

## いまやらないこと

- リール（動画制作コストが高い。カルーセルで十分な効果が出るまで着手しない）
- ストーリーズ（フォロワー100未満では効果薄）
- 広告出稿（オーガニックで伸びるか検証してから）
- インフルエンサー連携（Month 4+で検討）
- 施設情報の羅列だけの投稿（X戦略と同じ）
