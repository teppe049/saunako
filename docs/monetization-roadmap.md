# サウナ子 収益化ロードマップ（施設課金）

作成日: 2026-02-28

## 月収目標

| 時期 | 月間PV | 収益源 | 月収（保守的） | 月収（楽観的） |
|------|--------|--------|-------------|-------------|
| 3ヶ月後 | 5,000 | PR記事のみ | 3万円 | 5万円 |
| 6ヶ月後 | 10,000 | PR記事 + PR枠 | 8万円 | 15万円 |
| 1年後 | 20,000 | PR枠 + 送客手数料 | 20万円 | 40万円 |
| 2年後 | 50,000 | 送客 + PR枠 + 予約手数料 | 50万円 | 100万円 |

---

## Phase 1: 計測基盤を仕込む（今〜3ヶ月）

**目標: PV 5,000、送客データの蓄積**

| やること | 内容 |
|---------|------|
| `/go/[facilityId]` リダイレクト | クリック数をサーバー側で正確に計測 |
| UTMパラメータ付与 | 施設のGA4でも「サウナ子経由」が見える |
| SEO記事の継続 | トラフィックの土台 |

この段階での収益: ゼロ。ひたすらデータを溜める。

---

## Phase 2: 送客実績で営業開始（3〜6ヶ月）

**目標: PV 10,000、施設3〜5件と取引開始**

| やること | 単価 | 月収目安 |
|---------|------|---------|
| 施設にPR記事を提案 | 1本 3〜5万円 | 3〜5万円 |
| 送客レポートをメールで施設に共有 | 無料（信頼構築） | — |

### 営業の流れ

```
送客データを見せる
  →「先月 御施設に120クリック送客しました」
  →「PR記事を書きませんか？検索上位の当メディアで紹介します」
```

PR記事は施設側がお金を出しやすい形。「広告費」ではなく「取材・紹介記事」の体裁なので心理的ハードルが低い。

---

## Phase 3: PR枠の導入（6ヶ月〜1年）

**目標: PV 15,000〜、月収10〜20万円**

| 枠 | 内容 | 月額 |
|----|------|------|
| 検索結果の上位固定 | エリア検索で常に1〜2番目に表示 | 5,000〜10,000円 |
| トップページ「注目の施設」 | 全訪問者に露出 | 10,000円 |
| 施設ページ強化 | 写真追加枠・動画・クーポン表示 | 5,000円 |

### 料金の根拠

```
「PR枠に出すと月N回表示、うちM件が予約ページへクリックしています」
  → Phase 1 で溜めたデータがここで活きる
```

5〜10施設 × 月5,000〜10,000円 = 月5〜10万円
+ PR記事 月1〜2本 = 月5〜10万円

---

## Phase 4: 送客手数料の導入（1年〜）

**目標: PV 30,000〜、月収30〜50万円**

| モデル | 内容 | 月収目安 |
|-------|------|---------|
| クリック課金 | 予約ページへの送客1件あたり200〜500円 | 20〜40万円 |
| PR枠（継続） | Phase 3 の拡大 | 10〜15万円 |

Phase 3 のPR枠で関係ができた施設から、従量課金に切り替え提案。

```
「月額固定より、送客した分だけの方がリスクないですよ」
  → 施設にとっても成果報酬の方が受け入れやすい
```

---

## Phase 5: 予約のプラットフォーム化（2年〜）

| モデル | 内容 |
|-------|------|
| サウナ子上で予約完結 | 予約手数料 5〜10% |
| 施設向け管理画面 | 空き状況管理・送客分析・レビュー管理 |

---

## 収益の積み上がり

```
今          計測基盤（/go/ + UTM）
              ↓ データを溜める
3ヶ月後     PR記事で最初の売上（月3〜5万円）
              ↓ 施設との関係構築
6ヶ月後     PR枠導入（月10〜20万円）
              ↓ 送客データが十分に溜まる
1年後       送客手数料（月30〜50万円）
              ↓
2年後〜     予約プラットフォーム化（月100万円〜）
```

## 競合参考

- **サウナイキタイ**: 社員ゼロで売上7,000万円/年。ファン課金(月額370円×6,600人)+企業コラボ
- **食べログ**: 月額掲載料(1〜10万円/店) + 送客手数料(ランチ110円/人、ディナー220円/人)
- **kimoty.com(SISU)**: メディアは集客装置。本業のサウナ設備販売(数十万〜数百万円)に送客
