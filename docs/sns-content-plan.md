## 1. 投稿タイプ定義

| タイプ | 内容 | 写真 | 頻度目安 |
|--------|------|------|----------|
| **施設紹介** | 写真3-4枚 + 施設名・エリア・料金帯・特徴 + サイトリンク | 必須 | 週1-2 |
| **サウナ豆知識** | ロウリュのコツ、水風呂の入り方、整うメカニズム等 | 任意 | 週1 |
| **ランキング・比較** | 「水風呂が冷たいサウナTOP5」「コスパ最強3選」等 | あると◎ | 隔週1 |
| **新着・更新** | 新規掲載施設、料金変更、記事公開のお知らせ | 任意 | 随時 |
| **中の人・雑談** | サウナ子の日常、サウナあるある、季節ネタ | 任意 | 週0-1 |

## 2. 週間カレンダー

| 曜日 | 時間 | 投稿タイプ | 理由 |
|------|------|-----------|------|
| 火 | 21:00 | サウナ豆知識 or 中の人 | 週前半の軽いコンテンツ |
| 木 | 21:00 | 施設紹介 | 木曜夜はXエンゲージメント最大（週末前心理） |
| 金 | 21:00 | ランキング・比較 or 施設紹介 | 週末のサウナ来店意欲に直結 |
| 日 | 20:00 | 新着・雑談 | サウナ帰りユーザーが多い時間帯 |

- 週3の場合は日曜をスキップ
- 祝日前夜は施設紹介を優先（来店意欲が高い）

## 4. ハッシュタグ戦略

### 基本セット（全投稿共通）

```
#個室サウナ #サ活
```

**2タグを基本とし、最大3タグまで。** Xのアルゴリズムはタグ大量使用をスパム判定するため。

### 投稿タイプ別の追加候補（3つ目として1つ選ぶ）

| タイプ | 追加候補 |
|--------|---------|
| 施設紹介 | `#ととのった` `#{エリア名}サウナ` |
| 豆知識 | `#サウナ好きと繋がりたい` |
| ランキング | `#サウナ` |
| 雑談 | `#サウナー` `#ととのい` |

### 使わないタグ

- `#PR` `#AD`（ステマ誤認を避ける。広告でないなら付けない）
- 5個以上のタグ（スパム判定リスク）
- 無関係なトレンドタグ

## 5. 運用ガイドライン

### 投稿時間

- **基本: 21:00前後**（サウナ帰りユーザー＋全年代アクティブのゴールデンタイム）
- 日曜のみ **20:00**（早めの帰宅後を狙う）
- 祝日前夜は通常より1時間早め（20:00）も検討

### エンゲージメント

- `#個室サウナ` `#サ活` の投稿を**週2-3回RT・引用RT**
- サウナ系アカウント（施設公式、サウナメディア、サウナー）への**リプライ・引用RT**で認知拡大
- フォロワーからのリプライには**24時間以内に返信**（サウナ子口調で）

### UGC活用

- 個室サウナ体験を投稿しているユーザーを引用RTで紹介
- 「#サウナ子と整う」タグを将来的に導入（Phase 1、フォロワー300+で開始）

### UTMパラメータ

すべてのリンクに以下を付与:
```
?utm_source=x&utm_medium=social&utm_campaign={投稿タイプ}
```
- `utm_campaign` 例: `facility`, `tips`, `ranking`, `news`

### 禁止事項（character-guide.md 準拠）

- **敬語禁止** — 「おすすめです」→「おすすめだよ」
- **断定しすぎ禁止** — 「絶対行くべき」→「ぜひ行ってみてほしいな」
- **施設への否定的コメント禁止**
- **他施設との比較禁止** — 「AよりBのほうが〜」はNG
- **価格の具体的な言及禁止** — 「コスパがいい」程度に留める

### バナー画像生成

施設紹介やランキング投稿に画像を添付する場合、`/sns-banner` スキルで生成可能。
ブランドカラー・フォント・キャラクター素材は [sns-banner.md](../.claude/commands/sns-banner.md) を参照。
