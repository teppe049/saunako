**どんな人を探すか:**
- フォロワー100〜2,000くらい（お互い距離が近い規模感）
- bioにサウナ系の記載がある
- 最近も投稿している（アクティブ）
- いいねが少なめの投稿をしている（こちらのいいねに気づいてもらいやすい）

**いいねするときの感覚:**
- 24時間以内の投稿 → サウナでも日常でも、いいなと思ったらいいね
- 2-3日前のサウナ投稿 → まだいいねしても自然
- 1週間以上前 → さすがに不自然なのでスルー
- 愚痴やネガティブな内容 → スルー

**新しい人を見つけるコツ:**
- `#サ活` で検索すると定期的にサウナ報告してるコアなサウナ好きが多い
- `サウナ 行ってきた` は体験投稿が多くて、一般の人が見つかりやすい
- `#ととのう` はサウナ愛が強い人が使ってる
- `#個室サウナ` は施設アカウントの投稿がほとんどなので、一般の人は少なめ

## 気になる人リストの入れ替え

- 何日か続けていいねしても全く反応がない人は、無理に続けない
- フォローしてくれた人はリストから卒業（普通に交流を続ける）
- 新しく面白い人を見つけたら入れ替える
- 15人くらいをキープするのが見やすい

## フォローを返すかどうか

- サウナ好きの個人 → **返す**（仲間になれる）
- サウナメディア・ライター → **返す**（広げてもらえるかも）
- 個室サウナ施設 → **返す**（自分たちのサイトに載ってる施設だし）
- 大型サウナ（かるまる、サウナ東京とか） → **返さない**（向こうは気づかない）
- bot・スパム → **返さない**
- サウナイキタイ関連 → **返さない**（競合）
