- 1投稿1シーン（詰め込まない）
- ハッシュタグ: `#サウナ` `#個室サウナ` `#サウナ子`

## ターゲットリスト方式（フォロワー獲得の主軸）

### なぜターゲットリストか

- 単発いいね → 通知スルーされる
- 同じ人に3日連続いいね → 「この人誰？」とプロフィールを見に来る
- ランダムより反復のほうがフォロー転換率が高い

### ターゲット選定基準

| 条件 | 理由 |
|------|------|
| フォロワー100-2,000 | フォロー返しを期待できる規模 |
| bioにサウナ・サ活・ととのう等の記載あり | サウナ好きが確定 |
| 直近1週間以内に投稿 | アクティブ確認 |
| 投稿のいいね数が0-3 | いいねが目立つ（50%がこのゾーン） |
| 写真付きサ活報告をしている | 投稿意欲が高い＝リプに反応しやすい |

### ターゲット探索キーワード

| キーワード | 頻度 | 特徴 |
|-----------|------|------|
| `#サ活` | 高 | 定期的にサ活報告してるコア層 |
| `#サウナ` | 超高（数分に1件） | 母数が大きく候補を見つけやすい |
| `サウナ 行ってきた` | 高（1日数十件） | 体験投稿が多く、一般ユーザー率が高い |
| `#ととのう` | 中 | サウナ愛の強い層 |
| `#個室サウナ` | 低（1日10-15件） | 施設アカウントの投稿が大半。一般ユーザーは少ない |

※ 施設アカウントのリプ欄からの発掘は非効率（一般ユーザーのリプがほぼゼロ）

## サウナ好きの生態系（調査結果 3/12）

戦略の前提となるデータ。

### プロフィールの傾向

- 40%がbioに「サウナイキタイNo.XXXX」記載
- 30%が `#サ活` タグ入り
- 60%がサウナ+他趣味の列挙（銭湯/お酒/釣り等）
- 周辺ジャンル: 銭湯・温泉 > エンタメ（プロレス・アイドル・麻雀）

### いいね数の分布

- 0-3: 50%
- 3-10: 30%
- 10-30: 15%
- 30+: 5%

### 個室サウナへの温度感

- 大型施設・銭湯サウナ: 60%
- ホテル・スパ系: 20%
- 個室サウナ言及: 10%（ほぼ施設アカウント）
- 「行ってみたい」潜在層: 5-10%

→ サウナ好きの大半は大型施設派。個室サウナは「知ってるけど行ったことない」層が狙い目。
