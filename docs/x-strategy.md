## 投稿ルール

### トーン
- キャラクターガイド（docs/character-guide.md）に準拠
- 友達口調（だよ/かも/ね/よ）
- 敬語禁止、断定しすぎ禁止

### 形式
- 1投稿 140文字前後（長くても280文字以内）
- 絵文字: 👇等の機能的なもののみ。装飾系（✨🔥💧🌿💰📊等）は使わない
- URL はラスト行に置く
- 画像: 施設投稿には施設画像4枚を添付（`public/facilities/{id}-{0,1,2,3}.webp`）
- ハッシュタグ: `#個室サウナ` `#サウナ子` を全投稿に固定（テーマ別タグは付けない）

### 投稿時間
- **平日**: 12:00（昼休み）or 20:00（帰宅後）
- **金曜**: 18:00（週末計画タイム）
- **土日**: 10:00（朝のリサーチタイム）

## フォロー・エンゲージメント戦略

### フォロー方針

| フェーズ | アクション |
|---------|-----------|
| 初週 | 施設公式アカウントを50-100件フォロー |
| 2-3週目 | サウナ好き個人を1日10-20件フォロー |
| 1ヶ月目以降 | フォロー増やすよりエンゲージメントに集中 |

- 1日のフォロー上限: 40件（スパム判定回避）
- フォロー/フォロワー比率: 2:1 以下を目安
- フォロバ目的の大量フォロー→解除はやらない

### フォロー対象（優先順）

1. **個室サウナ施設の公式** — リポスト・引用RTのネタ元
2. **サウナ好き個人** — いいね・リプで存在アピール
3. **サウナメディア** — 業界動向キャッチ

### エンゲージメント

| アクション | 頻度 | 対象 |
|-----------|------|------|
| いいね | 1日20-30件 | 「個室サウナ」「プライベートサウナ」関連ツイート |
| リプ | 1日3-5件 | 「個室サウナどこがいい？」系に自然に回答 |
| 引用RT | 週2-3回 | 施設公式の投稿に一言添えて紹介 |
| リポスト | 週1-2回 | 施設の新オープン・キャンペーン情報 |

リプは宣伝臭を消す。「教えてあげる」スタンスで。

### X API 予算

- プラン: pay-per-use（従量課金）
- 投稿: $0.01/回
- 月間予算上限: **$5/月**

## 1ヶ月投稿カレンダー（3/11〜4/9）

毎日の投稿は `/x-post` スキルで生成する。
施設を取り上げる日は DeepResearch → データ差異チェック → 投稿文生成の流れ。

### 運用フロー

```
1. その日のテーマを確認（下のカレンダー参照）
2. `/x-post` を実行（施設IDを指定 or 自動選定）
3. DeepResearch結果 + データ差異レポートを確認
4. 差異があれば facilities.json を修正
5. 投稿文をコピペしてXに投稿
6. 投稿ログ（docs/x-post-log.md）に自動記録
```
