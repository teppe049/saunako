# サウナ子 X運用戦略（v2: 2026-03-14〜）

## ポジショニング

**本質**: 個室サウナを目的別に選べる案内アカウント
**入口**: デート軸（初めて個室サウナをデートで使いたい人）

「個室サウナ紹介アカウント」ではなく「選び方の案内人」。
打ち出しはデート、設計思想は目的別。後から初心者・女性・価格軸に広げられる土台にする。

### 競合との差別化
- サウナイキタイ: 総合サウナ検索（大型施設中心）
- サウナ子: 個室サウナ × 目的別（「誰向けか」が分かる）

## アカウント情報

- URL: https://x.com/saunako_jp
- サイト: https://www.saunako.jp

## 目的

| 優先度 | 目的 | 指標 |
|:------:|------|------|
| 1 | 認知 → フォロー | インプレッション → プロフ訪問 → フォロワー増 |
| 2 | フォロー → サイト流入 | リンククリック → GA4 Organic Social |

## ターゲット

**メイン**: サウナをデート選択肢として見始めた人（サウナガチ勢カップルではない）
**サブ検証**: 初心者 / 女性1人利用 / 価格重視

メインターゲットの特徴:
- 「個室サウナデート、気になるけどどこ行けばいいかわからない」
- 不安や疑問が多い → "案内人"の価値が出る
- サイトで条件検索する理由が生まれる

## 投稿カテゴリ

### 1. 認知投稿 — 非フォロワーに届ける

**型A: 条件別まとめ**
```
東京で「初デートにちょうどいい」個室サウナ、5つ選んでみた

選んだ基準👇
・駅チカ（迷わない）
・受付の動線がわかりやすい
・サウナ後にご飯行ける立地

#個室サウナ #サウナデート
```

**型B: 意外性・数字フック**
```
個室サウナ、カップルで行けるとこ実は全国に300以上あるって知ってた？

しかも半分は5,000円以下。
映画+ディナーより安いかも

#個室サウナ #サウナデート
```

### 2. 比較投稿 — 会話・プロフ訪問を増やす

**型C: 選び方の価値観提示**
```
初デートで個室サウナ行くなら、サウナの温度より「待合スペースの雰囲気」で選んだほうがいいと思う

サウナ出た後にリラックスして話せる場所があるかどうかで、その後の時間の質が全然変わるから
```

**型D: 二択・問いかけ**
```
デートで個室サウナ行くとして、

A: 完全個室（2人だけの空間）
B: 半個室（開放感あり、少しカジュアル）

みんなどっち派？
```

### 3. 信頼投稿 — 専門性を印象づける

2種類に分ける:

**型E: 体験レビュー（実際に行った時のみ）**
```
○○、行ってきたのでちょっとだけ正直に

良かったとこ
→ 水風呂の温度が自分で変えられる。デートだと相手に合わせられて最高

気になったとこ
→ 受付からサウナまでの導線がちょっとわかりにくい

向いてる人
→ サウナ好き同士のカップル
```

**型F: 選び方知識（行かなくても成立）**
```
個室サウナに初めて行く前に知っておきたいこと3つ

1. 水着いるかどうかは施設による（タオル貸出のとこも多い）
2. 予約制がほとんど。当日ふらっとは難しい
3. 「セルフロウリュ」はストーブに水かけるやつ。怖くない

#個室サウナ #サウナ初心者
```

### 4. 送客投稿 — サイトへの導線

**基本方針**: 単独カテゴリとしてではなく、他カテゴリの末尾に自然に混ぜる。

混ぜ方の例:
- 認知投稿の最後に「条件で探したい人はプロフへ」
- 比較投稿の最後に「似た施設まとめてます」
- 明示的な送客投稿は月2-3回に留める

**型G: 自然な導線（たまに単独投稿）**
```
デート向きの個室サウナ、条件で探せるページ作ったよ

「カップルOK」「駅チカ」「5,000円以下」とかで絞れる
気になる人はプロフのリンクからどうぞ👇
```

## 投稿ルール

### トーン
- キャラクターガイド（docs/character-guide.md）に準拠
- 友達口調（だよ/かも/ね/よ）
- 敬語禁止、断定しすぎ禁止
- AIっぽい無機質さを排除
- 毎回同じテンプレにならないように型を回す

### 形式
- **文字数**: 280ウェイト以内（日本語1文字=2, ASCII=1, 絵文字=2, 改行=1）
- 絵文字: 1-3個（機能的なもののみ。装飾系は使わない）
- **本文にURLを入れない**（Xアルゴリズムが外部リンク付き投稿のimpを下げる）
- URLは**リプライで分離投稿**する
- ハッシュタグ: `#個室サウナ` 固定 + テーマ別1個（`#サウナデート` `#サウナ初心者` 等）
- 画像: 施設投稿のみ4枚添付（`public/facilities/{id}-{0,1,2,3}.webp`）

### 投稿頻度
- **1日1本の本命投稿**（カテゴリを明示）
- 補助投稿: 0-1本（リプ or 引用RT）
- 日曜は休んでもOK

### 投稿時間
- 気が向いた時に投稿。時間帯の最適化はフォロワー100+になってから検討

## リプ運用

### 方針
- 露骨な「リプ配り」はしない
- 「最高ですね！」のような薄いリプを量産しない
- 目的は専門性の印象づけ

### やるリプの方向性
- 比較軸を足す
- 利用シーンを足す
- 初心者視点・女性視点・デート視点を足す
- 相手投稿の続きを自然に作る

### リプの例
- 「ここ、初心者でも入りやすい雰囲気ですか？」
- 「デートなら待合スペースの感じも気になります」
- 「個室で外気浴しっかり取れるの珍しいですね」

## いいね戦略

### 目的
フォロワー0からの立ち上げ。同じ人に繰り返しいいね → プロフィール訪問 → フォローの導線。

### 1日のルーティン（合計10分）

| やること | 対象 | いいね数 | 所要時間 |
|---------|------|:-------:|:-------:|
| ターゲットリストの新着チェック+いいね | 15人 | 7-10 | 5分 |
| 掲載施設アカウントの新着にいいね | アクティブ5施設 | 3-5 | 3分 |
| 大型施設の写真投稿にいいね | 1-2施設 | 1-2 | 1分 |
| 自分への反応チェック | — | — | 1分 |

### ターゲットリスト方式

詳細は `docs/x-like-strategy.md` を参照。

- 15人をキープ
- 1人あたり1日1-2いいね
- 3日連続反応なし → 入れ替え
- 週1で棚卸し

## KPI

### 日次（見るだけ、方針は変えない）
- インプレッション
- プロフィール訪問
- 新規フォロワー
- リンククリック

### 週次（ここで改善判断）
週次レビュー（`docs/x-weekly-review.md`）で以下を分析:
- 伸びた投稿 TOP3 / 伸びなかった WORST3
- カテゴリ別パフォーマンス
- 切り口別パフォーマンス（デート/初心者/価格/地域）
- タイトルの型（○選/○つの理由/知ってた？）
- CTA（保存訴求/質問/プロフ誘導）
- 来週増やす/減らすテーマ

### 月次（方向性を見直す）
- アカウントの方向性
- ターゲットの絞り方
- プロフィール・固定ポスト
- サイト導線

### 計算指標
- プロフ訪問率 = プロフ訪問 ÷ imp
- フォロー転換率 = 新規フォロワー ÷ プロフ訪問
- リンククリック率 = クリック ÷ プロフ訪問
- サイト詳細到達率 = 施設詳細閲覧 ÷ サイト流入
